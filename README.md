# Consulta-de-emprestimo
Simulador de análise de crédito em Python. O programa recebe os dados do cliente, calcula a prestação e decide se o empréstimo é aprovado ou negado, garantindo que a parcela não ultrapasse 30% da renda. Projeto prático focado em lógica matemática, condicionais (if/else) e manipulação de strings.
